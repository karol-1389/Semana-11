Arquitectura de Linux

Componentes principales:

  • Kernel
  • System Libraries
  • System Tools
  • Application Programs

Estructura general:

El sistema operativo Linux está estructurado en capas. En la parte más baja se encuentra el kernel, seguido de las bibliotecas del sistema que proporcionan funciones a nivel de sistema, luego las herramientas del sistema y, finalmente, las aplicaciones de usuario.

Kernel:

El kernel es el núcleo del sistema operativo. Administra los recursos del sistema, como la memoria, el procesador, los dispositivos de entrada/salida, y proporciona una interfaz para que otros programas accedan a estos recursos.

System Libraries:

Estas bibliotecas proporcionan funciones y procedimientos que los programas pueden utilizar para realizar tareas específicas, como la gestión de archivos, la conectividad de red, etc.

System Tools:

Las herramientas del sistema son programas que permiten a los usuarios y administradores interactuar con el sistema operativo. Estas incluyen comandos en la terminal, herramientas de gestión de archivos, herramientas de red, etc.

Application Programs:

Las aplicaciones de usuario son programas que utilizan los usuarios para realizar tareas específicas, como navegadores web, editores de texto, reproductores multimedia, entre otros.
